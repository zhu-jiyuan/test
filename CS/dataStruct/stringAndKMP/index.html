# 子串匹配问题

子串匹配又叫模式串匹配

例如 ab 是 sdabc 的子串， ab 又称为模式串， sdabc 又成为主串。

怎么样判断一个字符串是另一个字符串的子串，在文档中的查找、匹配等待都会用到。

# BF

利用滑动窗口，一步一步与主串进行比较，不对，让主串后移一位，重新与模式串进行比较。直到模式串完全匹配

时间复杂度 O (mn)
